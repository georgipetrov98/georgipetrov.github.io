Flower Shop

  • HTML & CSS
  • JavaScript
  • Bootstrap
  • C#
  • ASP.NET
  • PHP
  • MySQL
About

For the design of this project I used HTML, CSS and Bootstrap. For the backend PHP, ASP.NET, Mysql. The idea of this website is a Flower shop. Project comes with the option to log-in as an administrator and a guest. Administrators have different priveleges, whilst the guests are far more limited. As a guest you can buy flowers, request invoices, etc. But as an administrator you can do a lot more, like checking product stock, changing product pictures, deleting pictures and much more.

Student Management System

  • Java
  • Object-Oriented
  • Classes
  • UI/UX
About

For this project I used Java. This project is meant to depict a Student Management System. It was built with java and uses Objects. It has a Student List that displays all the students and data about them like names,grades,etc. You as a lecturer have options to manipulate that data like adding next students, changing their names, groups and other. There are also some quality of life functions like sorting the tables, updating them, seriliazing them to a file. You can also import Excel spreadsheets of students and Bulk assign students to groups and create new groups.
